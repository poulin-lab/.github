This repository contains organization-wide configuration files for the Poulin Lab GitHub. It is a special repo that is recognized by GitHub to display some files on the homepage of the organization. Nothing here should be modified regularly, it mainly stores shared templates and reference documents. **Please do not upload project-specific code here.**

---

## Contents

### 1. `profile/`
This folder controls the **public landing page** of the Poulin Lab GitHub organization.

- `profile/README.md`  
  Appears on the organization's main page when anyone visits.  
  *(Do not remove or rename this file.)*

---

### 2. Lab Documentation
- **Poulin_Lab_GitHub_Master_Doc.pdf**  
  A detailed guide for lab members on how to:
  - join GitHub  
  - upload and share code  
  - organize repositories  
  - use branches, commits, and pull requests  

Lab members who are new to GitHub should start with this document.
